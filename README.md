# RuPay Luxe Rewards Program

Welcome to the **RuPay Luxe Rewards Program**! This application provides an interactive platform for users to explore different card tiers and check their eligibility based on their financial profile.

## Features

- **Card Tiers**: Discover various card tiers with enhanced benefits.
- **Eligibility Calculator**: Input your financial details to see which card you qualify for.
- **Program Details**: Learn about the rewards and redemption options available.

## Installation

To run this application, you need to have Python installed along with the following packages:

```bash
pip install streamlit pandas
```

## Running the Application

1. Clone this repository:
   ```bash
   git clone <repository_url>
   cd <repository_directory>
   ```

2. Run the Streamlit application:
   ```bash
   streamlit run app.py
   ```

3. Open your web browser and navigate to `http://localhost:8501`.

## Usage

- **User Profile**: Users can input their CIBIL score, card experience, occupation, and annual income.
- **Eligibility Calculator**: Based on annual spending, the application calculates eligibility for different card tiers.
- **Program Details**: Users can view the benefits of the RuPay Luxe Rewards Program.
